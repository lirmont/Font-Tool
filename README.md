# Font Tool and Documentation

## Overview

![In-App Editing](https://raw.githubusercontent.com/lirmont/Font-Tool/master/Documentation/Media/Example-Usage.png)

Create image-based fonts from regular fonts or supply your own images. Effects are applied in divided hierarchies.

![Unicode Coverage](https://raw.githubusercontent.com/lirmont/Sprite-Animator/master/Documentation/Media/Unicode-Coverage-Report-Viewer-Scripts.png)

Among other things, check Unicode coverage of your font.

[Test Reel](https://youtu.be/2oAu_8RhC2E)

A 3D font, traced, bulked up, and exported for use in realtime by this program.

## Download Program/Pre-Installation

Font Tool 1.0.0.3: [ZIP file](https://drive.google.com/open?id=0B5iSdBA1bVSZb0JJMHhJVmlSUzA)

Install Panda3d (if you intend on exporting to EGG/BAM format): [Panda3d SDK](https://www.panda3d.org/download.php?sdk)

## Pre-Build

Install AForge: [AForge](http://www.aforgenet.com/framework/downloads.html)

Install SharpZipLib: [SharpZipLib](https://icsharpcode.github.io/SharpZipLib/)

Install Lumenworks Framework: [Lumenworks](https://www.nuget.org/packages/LumenWorksCsvReader/)

Install Newtonsoft.Json: [Newtonsoft.Json](http://www.newtonsoft.com/json)

Install OpenTK 1.1 (OpenGL wrapper): [OpenTK](https://sourceforge.net/projects/opentk/)
